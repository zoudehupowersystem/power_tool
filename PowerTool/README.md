# PowerTool（内层软件边界）

该目录用于标识“内层独立软件（PowerTool）”边界。当前仓库仍保持兼容布局（核心模块位于仓库根目录），本目录提供统一入口与清单，便于后续逐步迁移到明确的内外层结构：

- **内层（PowerTool）**：GUI、Python 内核计算、参数校核、结果解释。
- **外层（Agent/Skill）**：自然语言编排、skill 封装、报告与自动化流程。

## 入口
- `main.py`：调用仓库根目录 `power_tool.main()` 启动 GUI（兼容现有用法）。

## 核心清单
- `core_manifest.json`：列出内层软件的核心模块清单。

## 说明
本目录不改变现有导入路径与用户习惯；其目标是先建立架构边界，再逐步迁移文件位置。
