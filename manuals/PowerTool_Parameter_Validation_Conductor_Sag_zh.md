# PowerTool 导线弧垂计算

## 1. 功能定位
本页用于单档输电导线的工程近似弧垂计算。模型采用**不等高挂点悬链线**，并把**温度变化**与**载流量引起的温升**都折算到弧垂—张力关系里，适合做方案比较、教学演示和运行边界的快速判断。

它不是施工放线计算书，也不替代精细弧垂表、长期蠕变校核、风偏覆冰校核或 IEEE 738 级别的热平衡计算。

## 2. 页面结构
左侧输入区分为三块：

1. **几何与机械参数**：档距、左右挂点高度、单位质量、截面积、等效弹性模量、线膨胀系数、参考温度、参考水平张力。
2. **载流量—温度近似模型**：环境温度、20°C 电阻、电阻温度系数、等效冷却系数、太阳热增益。
3. **交互驱动变量**：导线温度滑块、载流量滑块，以及“按导线温度直接驱动 / 按载流量估温驱动”切换。

右侧显示关键指标卡、输电导线简图与悬链线图，以及结果说明文本。

## 3. 计算逻辑
### 3.1 悬链线几何
程序按单档、均布自重荷载的悬链线模型计算：
- 支持左右挂点不等高；
- 输出最大弦垂、跨中弦垂、跨内最小净空、左右挂点张力；
- 图中同时显示**参考状态导线**和**当前状态导线**，便于比较。

### 3.2 温度—张力耦合
程序以用户给定的 `T_ref` 与 `H_ref` 为参考状态，用“热膨胀 + 平均张力弹性伸长”的工程近似回算当前温度下的水平张力。

这意味着它适合快速比较：
- 温度升高后弧垂如何增大；
- 同一导线在不同挂点高差下的净空变化；
- 参考张力选取对运行弧垂的影响。

### 3.3 电流—温度近似
在“按载流量估温驱动”模式下，程序先按简化稳态热平衡

`I²R(T) + q_s = k_c (T_c - T_a)`

估算导线温度，再把该温度送入弧垂计算。这里：
- `R(T)` 为温度相关电阻；
- `q_s` 为等效太阳热增益；
- `k_c` 为等效冷却系数。

这个模型故意做得较轻，用于滑块交互与趋势判断；它不等价于 IEEE 738。

## 4. 推荐使用流程
1. 先输入档距、挂点高度和参考张力，确认参考工况合理。
2. 若已知导线运行温度，选择“按导线温度直接驱动”，拖动温度滑块查看弧垂变化。
3. 若更关心负荷电流影响，选择“按载流量估温驱动”，拖动载流量滑块观察温度与弧垂同步变化。
4. 重点关注以下指标：
   - 最大弦垂；
   - 跨内最小净空；
   - 当前水平张力；
   - 与参考状态相比的弧垂增量。

## 5. 输入建议
- **单位质量**：采用导线本体质量；若需要考虑覆冰，应先把附加荷载折算进去，但当前页面未显式区分风/冰来源。
- **等效弹性模量**：建议按导线型式和厂家资料填写；若不确定，可先用典型值做灵敏度比较。
- **参考水平张力**：尽量对应某一已知工况，例如 15°C 无风无冰张力或 20°C 初始张力。
- **等效冷却系数**：这是集总近似参数，建议结合经验、历史运行温升或已有热稳定结论回填，而不要把它当作精确气动参数。

## 6. 结果解释
- **最大弦垂**：导线相对两挂点连线的最大下垂量，是图中最直观的“弧垂”。
- **跨中弦垂**：跨中位置的弧垂，不一定等于最大弦垂，尤其在两挂点不等高时。
- **最小净空**：跨内导线最低处到地面的高度，是运行安全更直接的量。
- **水平张力**：温度升高通常会降低水平张力并增大弧垂。

## 7. 局限性
本页当前未显式考虑：
- 风偏；
- 覆冰与不均匀附加载荷；
- 长期蠕变；
- 分裂导线子导线间次级力学效应；
- 杆塔变形与绝缘子串偏转；
- 更高精度的热平衡与辐射/对流细节。

因此，本页结果适合做**工程近似判断**，不应用作施工放线或正式定值文件的唯一依据。
