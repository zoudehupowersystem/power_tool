# 短路电流计算使用手册（详细版）

## 1. 功能
计算三相、两相、单相等故障工况下的短路电流与电压分布，并提供向量图辅助判断。

## 2. 输入说明
- 系统电压、线路参数、零序参数、接地电阻、故障电阻需统一单位。
- 故障点位置按线路百分比输入。

## 3. 推荐步骤
1. 先做三相短路建立上限基线。
2. 再做单相接地评估最不利接地条件。
3. 对比电流图与电压图，判断保护动作需求。

## 4. 结果判读
- 电流峰值用于校核开断能力。
- 电压跌落分布用于评估电压敏感负荷影响。

## 5. 常见错误
- 忽略零序路径导致单相故障结果失真。
- 中性点电阻口径与现场配置不一致。

## 6. 建议
将本页结果与保护定值、开关设备能力联合校核。

## 附录A：参数核对清单
- 是否使用统一容量基准与电压基准。
- 是否确认输入量纲（pu、kV、MVA、Ω、秒）无误。
- 是否核查关键设备参数来源（铭牌/试验报告/调度台账）。
- 是否在高峰、平峰、低谷三个典型工况做过对比。
- 是否记录“输入参数—结论—建议动作”的闭环表格。

## 附录B：结果落地建议
1. 先把页面结果用于“快速筛选方案”。
2. 对筛选出的可行方案，建立复核算例（至少 3 组边界工况）。
3. 与保护、调度、运维联合审阅关键结论。
4. 形成可执行动作表：动作条件、执行步骤、回退条件、告警阈值。
5. 上线前在正式仿真平台做最终验证并归档。

## 附录C：常见问答（FAQ）
- **问：为什么页面结果与现场记录有偏差？**
  - 答：本软件是工程近似工具，未覆盖所有拓扑细节、控制死区和保护逻辑，建议将结果作为方向性参考。
- **问：如何提高结果可信度？**
  - 答：优先保证输入质量与基准一致性，再用多工况对比验证结论稳定性。
- **问：什么时候必须转正式仿真？**
  - 答：当涉及投运定值、保护整定、重大运行方式切换或安全边界接近时，应转入正式仿真流程。

## 附录D：示例工况记录模板
| 项目 | 建议填写内容 |
|---|---|
| 工况名称 | 如“夏季晚高峰、主变N-1方式” |
| 关键输入 | 电压、潮流、设备容量、等值参数 |
| 页面结论 | 稳定性判断/分区判断/风险等级 |
| 建议动作 | 调档、补偿投切、运行方式调整 |
| 复核结论 | 是否需要进入正式潮流/暂稳仿真 |

## 附录E：交付建议
- 输出给运行人员时，建议附上“适用边界”与“禁止条件”。
- 输出给调试人员时，建议附上“参数来源表”与“版本号”。
- 输出给管理人员时，建议附上“风险等级”和“执行成本”。
- 每次使用后保留截图与文本结果，便于事件复盘。

## 附录F：复盘建议
建议在每次分析后，至少记录：输入版本、运行方式、关键判断、执行动作、执行后反馈。累计样本后可形成本单位自己的经验曲线与参数库，提高后续分析效率与一致性。
