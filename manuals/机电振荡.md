# 机电振荡使用手册（工程版）

## 1. 页面目标
机电振荡页面用于估算低频振荡特性，包括振荡频率、阻尼趋势和功角摆动风险。它适合做方案初筛、参数灵敏度分析与教学展示。

## 2. 建模边界
本页采用简化等值模型，适用于“单通道主导机电振荡”的近似判断，不替代完整多机特征值分析与时域仿真。

## 3. 输入参数建议
- 内电势 `E'_q`、端电压 `U`、等值电抗 `X_Σ`：建议使用同一工况下折算值。
- 初始有功 `P0`：接近运行点，避免过度偏离导致解释失真。
- 惯性时间常数 `T_j`、同步频率 `f0`：用于确定振荡动态量级。

## 4. 使用流程
1. 输入额定工况，获取基线频率与阻尼结论。
2. 逐步改变 `X_Σ` 或 `P0`，观察振荡频率变化。
3. 对比不同惯性参数，识别“低惯性高风险”组合。
4. 形成“工况-模态趋势”笔记用于后续 SMIB 深化分析。

## 5. 结果判读
- 振荡频率升高并不代表更稳定，需要结合阻尼判断。
- 阻尼不足时，应结合 AVR/PSS、运行点、网络强度联合优化。
- 若结果边界敏感，建议进一步进入小扰动页面做模态分析。

## 6. 排错清单
- 核查输入是否存在负值或不合物理常识。
- 对照历史工况，确认参数是否为同一基准折算。
- 若和 PMU 数据差异大，考虑模型简化引起的系统性偏差。

## 附录A：参数核对清单
- 是否使用统一容量基准与电压基准。
- 是否确认输入量纲（pu、kV、MVA、Ω、秒）无误。
- 是否核查关键设备参数来源（铭牌/试验报告/调度台账）。
- 是否在高峰、平峰、低谷三个典型工况做过对比。
- 是否记录“输入参数—结论—建议动作”的闭环表格。

## 附录B：结果落地建议
1. 先把页面结果用于“快速筛选方案”。
2. 对筛选出的可行方案，建立复核算例（至少 3 组边界工况）。
3. 与保护、调度、运维联合审阅关键结论。
4. 形成可执行动作表：动作条件、执行步骤、回退条件、告警阈值。
5. 上线前在正式仿真平台做最终验证并归档。

## 附录C：常见问答（FAQ）
- **问：为什么页面结果与现场记录有偏差？**
  - 答：本软件是工程近似工具，未覆盖所有拓扑细节、控制死区和保护逻辑，建议将结果作为方向性参考。
- **问：如何提高结果可信度？**
  - 答：优先保证输入质量与基准一致性，再用多工况对比验证结论稳定性。
- **问：什么时候必须转正式仿真？**
  - 答：当涉及投运定值、保护整定、重大运行方式切换或安全边界接近时，应转入正式仿真流程。
