# 录波曲线使用手册（详细版）

## 1. 功能
支持 COMTRADE、Yokogawa、MATLAB 录波文件加载，通道浏览、时间窗缩放、序分量与频谱分析，并支持按通道“重新导出”。

## 2. 支持格式
- COMTRADE：CFG + DAT（ASCII/BINARY）
- Yokogawa：WVF/WDF（含配套头信息）
- MATLAB：MAT（优先兼容 MAT v5，兼容本工具历史 MAT v4 文件）
- 文本编码支持 UTF-8 与 GB2312

## 2.1 重新导出（新增）
1. 加载录波后点击“重新导出”。
2. 在弹窗中可勾选一个或多个通道。
3. 选择导出格式：COMTRADE / CSV / MATLAB（三选一）。
4. 设置输出路径与文件名后执行导出。

说明：
- 导出为 CSV、MATLAB 时，直接写入物理量值（无需关心整数码值或比例系数）。
- 导出文件会保留通道名；COMTRADE 还会同步生成 CFG + DAT 两个文件。

## 3. 推荐流程
1. 先加载文件并检查通道列表。
2. 设置时间窗，定位故障前后关键区间。
3. 执行序分量与频谱分析，提取关键特征。

## 4. 结果解释
- 三相不平衡可优先看负序、零序分量。
- 谐波异常可结合工况切换与设备动作时间点判断。
- “分析选中通道”会额外给出：最小值/最大值及发生时刻、非周期分量（恒定直流、衰减直流、衰减时间常数）。
- 点击“分析选中通道”后会弹出谐波可视化窗口，包含谐波含有量表格与含有率柱状图，最高次数可调。

## 5. 常见问题
- 通道名乱码：优先检查原始文件编码与配套文件完整性。
- 时间轴异常：确认 DAT 采样率与 CFG 描述一致。

## 6. 建议
分析结论应与保护动作记录、SCADA 事件时标交叉校验。

## 附录A：参数核对清单
- 是否使用统一容量基准与电压基准。
- 是否确认输入量纲（pu、kV、MVA、Ω、秒）无误。
- 是否核查关键设备参数来源（铭牌/试验报告/调度台账）。
- 是否在高峰、平峰、低谷三个典型工况做过对比。
- 是否记录“输入参数—结论—建议动作”的闭环表格。

## 附录B：结果落地建议
1. 先把页面结果用于“快速筛选方案”。
2. 对筛选出的可行方案，建立复核算例（至少 3 组边界工况）。
3. 与保护、调度、运维联合审阅关键结论。
4. 形成可执行动作表：动作条件、执行步骤、回退条件、告警阈值。
5. 上线前在正式仿真平台做最终验证并归档。

## 附录C：常见问答（FAQ）
- **问：为什么页面结果与现场记录有偏差？**
  - 答：本软件是工程近似工具，未覆盖所有拓扑细节、控制死区和保护逻辑，建议将结果作为方向性参考。
- **问：如何提高结果可信度？**
  - 答：优先保证输入质量与基准一致性，再用多工况对比验证结论稳定性。
- **问：什么时候必须转正式仿真？**
  - 答：当涉及投运定值、保护整定、重大运行方式切换或安全边界接近时，应转入正式仿真流程。

## 附录D：示例工况记录模板
| 项目 | 建议填写内容 |
|---|---|
| 工况名称 | 如“夏季晚高峰、主变N-1方式” |
| 关键输入 | 电压、潮流、设备容量、等值参数 |
| 页面结论 | 稳定性判断/分区判断/风险等级 |
| 建议动作 | 调档、补偿投切、运行方式调整 |
| 复核结论 | 是否需要进入正式潮流/暂稳仿真 |

## 附录E：交付建议
- 输出给运行人员时，建议附上“适用边界”与“禁止条件”。
- 输出给调试人员时，建议附上“参数来源表”与“版本号”。
- 输出给管理人员时，建议附上“风险等级”和“执行成本”。
- 每次使用后保留截图与文本结果，便于事件复盘。
