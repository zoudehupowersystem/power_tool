# 小扰动分析使用手册（详细版）

## 1. 页面功能
用于 SMIB 小扰动分析，输出特征值、阻尼比、参与因子，并支持 AVR/PSS 参数影响评估。

## 2. 页面组成
- 工况与网络
- 六阶机组
- AVR III
- PSS II
- 1型 AVR/PSS

## 3. 推荐使用流程
1. 在“工况与网络”建立基线平衡点。
2. 在“六阶机组”核对关键参数（惯性、阻尼、时常数）。
3. 配置 AVR/PSS 参数并计算特征值。
4. 重点查看低阻尼模态的参与因子，确定主导状态量。

## 4. 结果解释
- 实部越靠近 0，稳定裕度越低。
- 阻尼比低的模态应优先治理。
- 参与因子可用于确定“该调谁最有效”。

## 5. 常见问题
- 参数来自不同机型模板，导致物理意义不一致。
- 仅看单个特征值，忽略模态簇变化。

## 6. 实际建议
先做灵敏度扫描，再做定点细调；保留每次调参记录，便于回溯。

## 附录A：参数核对清单
- 是否使用统一容量基准与电压基准。
- 是否确认输入量纲（pu、kV、MVA、Ω、秒）无误。
- 是否核查关键设备参数来源（铭牌/试验报告/调度台账）。
- 是否在高峰、平峰、低谷三个典型工况做过对比。
- 是否记录“输入参数—结论—建议动作”的闭环表格。

## 附录B：结果落地建议
1. 先把页面结果用于“快速筛选方案”。
2. 对筛选出的可行方案，建立复核算例（至少 3 组边界工况）。
3. 与保护、调度、运维联合审阅关键结论。
4. 形成可执行动作表：动作条件、执行步骤、回退条件、告警阈值。
5. 上线前在正式仿真平台做最终验证并归档。

## 附录C：常见问答（FAQ）
- **问：为什么页面结果与现场记录有偏差？**
  - 答：本软件是工程近似工具，未覆盖所有拓扑细节、控制死区和保护逻辑，建议将结果作为方向性参考。
- **问：如何提高结果可信度？**
  - 答：优先保证输入质量与基准一致性，再用多工况对比验证结论稳定性。
- **问：什么时候必须转正式仿真？**
  - 答：当涉及投运定值、保护整定、重大运行方式切换或安全边界接近时，应转入正式仿真流程。

## 附录D：示例工况记录模板
| 项目 | 建议填写内容 |
|---|---|
| 工况名称 | 如“夏季晚高峰、主变N-1方式” |
| 关键输入 | 电压、潮流、设备容量、等值参数 |
| 页面结论 | 稳定性判断/分区判断/风险等级 |
| 建议动作 | 调档、补偿投切、运行方式调整 |
| 复核结论 | 是否需要进入正式潮流/暂稳仿真 |

## 附录E：交付建议
- 输出给运行人员时，建议附上“适用边界”与“禁止条件”。
- 输出给调试人员时，建议附上“参数来源表”与“版本号”。
- 输出给管理人员时，建议附上“风险等级”和“执行成本”。
- 每次使用后保留截图与文本结果，便于事件复盘。
