# 参数校核与标幺值：两绕组变压器（详细手册）

## 1. 用途
用于两绕组主变参数折算、标幺化与越界校验。

## 2. 输入重点
- `SN/UN/Uk/Pk/I0/P0` 建议取同一试验报告数据。
- 基准容量/电压应与系统分析口径一致。

## 3. 使用步骤
1. 输入基础参数并计算。
2. 核查等值阻抗与励磁支路参数。
3. 对比典型范围提示，识别异常点。

## 4. 常见问题
- Uk% 与 Pk 来自不同温度条件。
- 额定容量误填导致标幺量级整体偏移。

## 5. 建议
将该页输出作为短路与稳态分析统一参数底稿。

- 是否使用统一容量基准与电压基准。

- 是否记录“输入参数—结论—建议动作”的闭环表格。

## 附录B：结果落地建议
1. 先把页面结果用于“快速筛选方案”。
2. 对筛选出的可行方案，建立复核算例（至少 3 组边界工况）。
3. 与保护、调度、运维联合审阅关键结论。
4. 形成可执行动作表：动作条件、执行步骤、回退条件、告警阈值。
5. 上线前在正式仿真平台做最终验证并归档。

## 附录C：常见问答（FAQ）
- **问：为什么页面结果与现场记录有偏差？**
  - 答：本软件是工程近似工具，未覆盖所有拓扑细节、控制死区和保护逻辑，建议将结果作为方向性参考。
- **问：如何提高结果可信度？**
  - 答：优先保证输入质量与基准一致性，再用多工况对比验证结论稳定性。
- **问：什么时候必须转正式仿真？**
  - 答：当涉及投运定值、保护整定、重大运行方式切换或安全边界接近时，应转入正式仿真流程。
