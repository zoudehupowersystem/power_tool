# 电压无功分析：静态电压稳定（详细手册）

## 1. 功能说明
本页面用于评估静态电压稳定裕度，帮助快速判断“当前无功支撑是否接近极限”。

## 2. 输入前准备
- 送端电压、线路/变压器等值电抗、功率因数、容量基准需统一折算。
- 建议先基于调度主方式或典型高峰工况输入。

## 3. 推荐操作步骤
1. 填入基线工况，记录页面给出的稳定性判断。
2. 逐步降低功率因数（加重无功需求），观察边界移动。
3. 对比不同 `X_Σ`（投切电抗器/改变运行方式）下的裕度变化。
4. 形成“边界点列表”，作为后续无功规划输入。

## 4. 结果解释
- 稳定裕度越小，表示越接近电压崩溃风险区。
- 低功率因数场景下，电压-无功耦合更敏感。
- 若判断接近边界，应结合 AVC/无功补偿页面制定动作策略。

## 5. 常见误区
- 仅凭单一工况做结论：建议至少覆盖峰荷、平荷、低谷三档。
- 只看电压不看无功：静稳问题本质是电压与无功共同约束。

## 6. 工程建议
- 本页可作为无功优化前置筛查。
- 对关键断面仍需配合连续潮流/QV 曲线分析复核。

## 附录A：参数核对清单
- 是否使用统一容量基准与电压基准。
- 是否确认输入量纲（pu、kV、MVA、Ω、秒）无误。
- 是否核查关键设备参数来源（铭牌/试验报告/调度台账）。
- 是否在高峰、平峰、低谷三个典型工况做过对比。
- 是否记录“输入参数—结论—建议动作”的闭环表格。

## 附录B：结果落地建议
1. 先把页面结果用于“快速筛选方案”。
2. 对筛选出的可行方案，建立复核算例（至少 3 组边界工况）。
3. 与保护、调度、运维联合审阅关键结论。
4. 形成可执行动作表：动作条件、执行步骤、回退条件、告警阈值。
5. 上线前在正式仿真平台做最终验证并归档。

## 附录C：常见问答（FAQ）
- **问：为什么页面结果与现场记录有偏差？**
  - 答：本软件是工程近似工具，未覆盖所有拓扑细节、控制死区和保护逻辑，建议将结果作为方向性参考。
- **问：如何提高结果可信度？**
  - 答：优先保证输入质量与基准一致性，再用多工况对比验证结论稳定性。
- **问：什么时候必须转正式仿真？**
  - 答：当涉及投运定值、保护整定、重大运行方式切换或安全边界接近时，应转入正式仿真流程。
